🌱 AgroSense AI
Smart Crop Disease Detection, Prediction & Prevention System
📌 Overview

AgroSense AI is an intelligent agriculture support system designed to assist farmers in identifying crop diseases at an early stage and taking the right preventive actions.

The system uses machine learning and image analysis to detect diseases from crop images and provides practical solutions instantly. It reduces dependency on manual inspection and helps farmers make faster, informed decisions.

🎯 Objective
To automate crop disease detection using AI
To provide instant and reliable prevention methods
To reduce crop loss and improve productivity
To make advanced technology accessible to farmers

⚙️ How to Run
1. Clone Repository

git clone https://github.com/your-username/AgroSense-AI.git
cd AgroSense-AI

2. Setup Backend

cd backend
pip install -r requirements.txt
python app.py

3. Setup Frontend

cd frontend
npm install
npm start

🧩 Problem Statement

Crop diseases are a major reason for reduced agricultural productivity.
Traditional detection methods are slow, manual, and depend heavily on experts.

Farmers, especially in rural areas, often lack timely access to proper guidance.

AgroSense AI solves this by providing fast, automated, and intelligent disease detection along with actionable solutions.
